# Busbar App

Local busbar generator with a FastAPI backend, HTML form, and Three.js 3D viewer.

## Requirements

- Python 3.11+
- CadQuery (system install — not listed in `requirements.txt`)

## Setup

```bash
pip install -r requirements.txt
```

## Run

```bash
python app.py
```

The app starts at [http://localhost:8000](http://localhost:8000) and opens your browser automatically.

## Output

Generated STEP and GLB files are written to `output/`.
